Foreign Whispers

Foreign Whispers is a project that develops a multilingual video processing system. It starts by downloading YouTube videos from the a channel using a Python API. The audio from these videos is then separated and converted to text. This translation process is designed to be flexible, allowing for a variety of languages. The translated text is then converted back into speech using another Python API. This creates a spoken version of the translated text, enhancing the accessibility of the content. Finally, all these components are integrated into a user-friendly interface. This interface allows users to input a YouTube video and receive the same video but with subtitles in their chosen language.
